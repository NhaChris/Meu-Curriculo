# Portfólio Front-End | ChrisDev

Este é um projeto desenvolvido com o objetivo de demonstrar minhas habilidades e capacidades técnicas no desenvolvimento **Front-End**.

O foco principal aqui é a aplicação de interfaces modernas, animações fluidas e uma experiência de usuário (UX) de alto nível, utilizando as tecnologias mais atuais do mercado.

## 🚀 Tecnologias Utilizadas

- **Next.js** (App Router)
- **React**
- **Tailwind CSS** (Estilização)
- **Framer Motion / GSAP** (Animações)
- **FontAwesome** (Ícones)
